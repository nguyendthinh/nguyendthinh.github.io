iSimon Says

My futuristic take on the classic Simon Says game where one must repeat exactly the sequence of colors that Simon says to repeat. I utilized JavaScript in conjunction with a jQuery library in order to successfully program the logic behind the game.


Hey, I'm Bored!

A fun application that aims at helping people find nearby events at a last-minute notice... or when they simply need their boredom relieve right away! This is a MEAN stack (MongoDB, Express, Angular, Node) application that utilizes a RESTful server and integration with a third party API that specializes in listing nearby local events.


Block Party

A color-based game that was created using React and vanilla JavaScript. The objective of the game is to get every block on the gameboard to be the same color. In order to do that, just click on a block and watch as the adjacent blocks change to the color of the block that you clicked on! But watch out, every 20 seconds the board will change a few blocks on the gameboard to a different color to try to mess you up. Win the game for a cool outcome!


Dance Website

A minimalistic website that I've made for a friend who is a dancer to showcase his unique personality as well as dancing chops. jQuery and Bootstrap were mostly implemented in the creation of this website.
